Playlist Data Converter

This repository contains a small automation tool that converts public playlist files (M3U format) into a structured JSON format for use in software testing, data processing, and application development.

The project focuses only on parsing and transforming text-based playlist metadata.

---

Important Notice

This repository does not host, store, or distribute any media content.

The software included here only:

- Reads playlist text files
- Extracts metadata such as name, logo, and URL
- Converts that information into a structured JSON format

All URLs contained in playlist sources remain hosted on external third-party servers.

This repository has no control over those servers or their content.
